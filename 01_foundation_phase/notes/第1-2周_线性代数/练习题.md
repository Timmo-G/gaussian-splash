### P1. 3DGS协方差矩阵推导

给定旋转角度 θ=45°（绕Z轴），缩放 s=(2.0, 0.5, 1.0)：

1. 构造旋转矩阵 R（3×3）
2. 构造缩放矩阵 S = diag(2.0, 0.5, 1.0)
3. 手推 Σ = R·S·Sᵀ·Rᵀ（展开每一步，不要跳步）
4. 验证 Σ 的特征值等于 sx²=4, sy²=0.25, sz²=1
5. 解释：为什么旋转不改变特征值？

```python
import numpy as np
theta = np.radians(45)
R = np.array([[np.cos(theta), -np.sin(theta), 0],
              [np.sin(theta),  np.cos(theta), 0],
              [0,              0,             1]])
S = np.diag([2.0, 0.5, 1.0])
Sigma = R @ S @ S.T @ R.T
print("特征值:", np.linalg.eigvalsh(Sigma))
```
