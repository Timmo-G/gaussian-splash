### P1. 雅可比矩阵完整推导

透视投影函数 π(X, Y, Z) = (fx·X/Z + cx, fy·Y/Z + cy)

1. 手推雅可比矩阵 J = ∂π/∂(X,Y,Z)（逐元素展开）
2. 验证：当 Z 很大时，J 的各元素趋向于什么？
3. 验证：当 Z 很小时，J 的各元素趋向于什么？（近处高斯投影更大）
4. 用数值差分验证你的雅可比矩阵

```python
def projection(mu_cam, fx, fy, cx, cy):
    x, y, z = mu_cam
    return np.array([fx*x/z + cx, fy*y/z + cy])

def numerical_jacobian(mu_cam, fx, fy, cx, cy, eps=1e-5):
    J = np.zeros((2, 3))
    for i in range(3):
        delta = np.zeros(3)
        delta[i] = eps
        J[:, i] = (projection(mu_cam+delta, fx, fy, cx, cy) -
                   projection(mu_cam-delta, fx, fy, cx, cy)) / (2*eps)
    return J
```
