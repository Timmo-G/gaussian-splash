## 0. 为什么高斯分布是3DGS的核心？

3D Gaussian Splatting 的名字直接来自"高斯分布"。每个3D高斯基元就是一个三维高斯分布：

```
N(x; μ, Σ) = 1/((2π)^(3/2)|Σ|^(1/2)) · exp(-1/2 · (x-μ)ᵀΣ⁻¹(x-μ))
```

**为什么选择高斯分布？**

1. **可微分**：高斯函数处处可微，梯度有解析表达式，支持梯度下降优化
2. **投影封闭性**：3D高斯投影到2D仍然是高斯（这使得渲染成为可能）
3. **参数少**：用均值μ和协方差矩阵Σ完全描述，参数数量可控
4. **几何直觉好**：等概率密度面是椭球体，符合建筑场景中平面结构的特点

## 3.1 概率基础

### 随机变量与概率密度函数（PDF）

**随机变量**：取值不确定的量。例如：
- 一次测量的误差
- 一个像素的颜色值
- 一个高斯基元的位置（在训练前是随机的）

**概率密度函数 p(x)**：描述随机变量在各个值附近的"密度"。

**三个基本性质**：
1. p(x) ≥ 0（非负）
2. ∫p(x)dx = 1（归一化）
3. P(a ≤ X ≤ b) = ∫ₐᵇ p(x)dx（区间概率）

### 期望与方差 —— 完整推导

**期望**（均值）：随机变量的"加权平均"
```
μ = E[X] = ∫ x·p(x) dx
```

**方差**：随机变量的"分散程度"
```
σ² = Var[X] = E[(X-μ)²] = E[X²] - μ²
```

**推导 Var[X] = E[X²] - μ²**：
```
Var[X] = E[(X-μ)²]
       = E[X² - 2μX + μ²]
       = E[X²] - 2μE[X] + μ²
       = E[X²] - 2μ² + μ²
       = E[X²] - μ²  ✓
```

**标准差** σ = √(Var[X])，与 X 同单位，更直观。

### 累积分布函数（CDF）

```
F(x) = P(X ≤ x) = ∫₋∞ˣ p(t) dt
```

CDF 是 PDF 的积分，单调递增，从0到1。

**关系**：p(x) = dF(x)/dx（PDF是CDF的导数）

## 3.2 一维高斯分布 —— 完整推导

### 概率密度函数

```
N(x; μ, σ²) = 1/(√(2πσ²)) · exp(-(x-μ)²/(2σ²))
```

**为什么有这个形式？**

高斯分布是最大熵分布（在均值和方差固定的条件下，熵最大的分布）。这意味着它是"最不确定"的分布，是自然界中最常见的分布形式。

**归一化系数 1/√(2πσ²) 的推导**：

设 I = ∫₋∞^∞ exp(-x²/2) dx，则 I² = ∫∫ exp(-(x²+y²)/2) dxdy

转换为极坐标：I² = ∫₀^∞ ∫₀^{2π} exp(-r²/2) r dθ dr = 2π ∫₀^∞ r·exp(-r²/2) dr = 2π

所以 I = √(2π)，即 ∫₋∞^∞ exp(-x²/2) dx = √(2π)

因此 ∫₋∞^∞ 1/√(2π) · exp(-x²/2) dx = 1 ✓

### 参数的几何意义

- **μ**（均值）：控制分布的中心位置（分布的"重心"）
- **σ**（标准差）：控制分布的宽窄（σ越大越宽，越"不确定"）
- **σ²**（方差）：σ的平方，描述分散程度

### 68-95-99.7 规则

| 范围 | 精确概率 | 近似 |
|------|---------|------|
| [μ-σ, μ+σ] | 68.27% | ~68% |
| [μ-2σ, μ+2σ] | 95.45% | ~95% |
| [μ-3σ, μ+3σ] | 99.73% | ~99.7% |

**3DGS中的意义**：高斯基元的"影响范围"通常取3σ以内，超出范围的贡献可以忽略（< 0.3%）。这是3DGS渲染时"提前终止"的依据。

### 高斯函数的可微性

高斯函数处处可微，梯度有解析表达式：

```
∂N/∂μ = N(x;μ,σ²) · (x-μ)/σ²
∂N/∂σ = N(x;μ,σ²) · ((x-μ)²/σ³ - 1/σ)
```

**3DGS中的意义**：正是因为高斯函数可微，才能对所有参数做梯度下降优化。

## 3.3 多维高斯分布

### 概率密度函数

```
N(x; μ, Σ) = 1/((2π)^(d/2)|Σ|^(1/2)) · exp(-1/2 · (x-μ)ᵀΣ⁻¹(x-μ))
```

其中 d 是维度，Σ 是协方差矩阵（d×d 正定对称矩阵）。

### 等概率密度面是椭球体 —— 推导

令指数部分等于常数 c²：
```
(x-μ)ᵀ Σ⁻¹ (x-μ) = c²
```

对 Σ 做特征分解：Σ = QΛQᵀ，则 Σ⁻¹ = QΛ⁻¹Qᵀ

令 y = Λ^(-1/2) Qᵀ (x-μ)（白化变换），则：
```
yᵀy = c²
```

这是一个球面！反变换 x = Q Λ^(1/2) y + μ，球面被 Q Λ^(1/2) 变换成椭球面。

椭球的：
- **轴方向** = Σ 的特征向量（Q 的列）
- **半轴长** = c × √(特征值) = c × √λᵢ

### 边缘分布：高维高斯投影仍是高斯

**定理**：如果 X = (X₁, X₂) 服从联合高斯分布，则 X₁ 的边缘分布也是高斯分布。

**证明**（简化版）：
```
p(x₁) = ∫ p(x₁, x₂) dx₂
```

对联合高斯分布积分，指数项中关于 x₂ 的部分可以配方，积分后仍然是高斯形式。

**3DGS中的意义**：3D高斯投影到2D图像平面后，仍然是2D高斯分布！这个性质使得可微分光栅化成为可能。

## 3.4 3D高斯投影到2D —— 完整推导

这是3DGS渲染的关键步骤，也是最重要的数学推导。

### 问题设置

给定：
- 3D高斯：均值 μ₃D（相机坐标系），协方差 Σ₃D
- 相机内参：焦距 fx, fy，主点 cx, cy
- 透视投影函数：π(X, Y, Z) = (fx·X/Z + cx, fy·Y/Z + cy)

求：2D投影高斯的均值 μ₂D 和协方差 Σ₂D

### 精确投影（为什么不用）

透视投影是非线性的（除以Z），所以3D高斯经过透视投影后，严格来说不再是高斯分布。

### 雅可比近似（3DGS的做法）

用一阶泰勒展开（雅可比矩阵）在均值处线性化投影函数：

```
π(x) ≈ π(μ₃D) + J · (x - μ₃D)
```

其中 J 是投影函数在 μ₃D 处的雅可比矩阵（2×3）。

**雅可比矩阵的推导**（逐元素展开）：

设 μ₃D = (X₀, Y₀, Z₀)，投影函数：
```
u = fx · X/Z + cx
v = fy · Y/Z + cy
```

对 X 求偏导：∂u/∂X = fx/Z₀，∂v/∂X = 0
对 Y 求偏导：∂u/∂Y = 0，∂v/∂Y = fy/Z₀
对 Z 求偏导：∂u/∂Z = -fx·X₀/Z₀²，∂v/∂Z = -fy·Y₀/Z₀²

所以：
```
J = [[fx/Z₀,    0,   -fx·X₀/Z₀²],
     [0,    fy/Z₀,   -fy·Y₀/Z₀²]]
```

### 投影协方差的推导

线性变换 y = Jx 的协方差变换公式：
```
Cov(y) = J · Cov(x) · Jᵀ
```

但3D高斯在世界坐标系中，需要先经过相机旋转 W = R_cam：

```
Σ₂D = J · W · Σ₃D · Wᵀ · Jᵀ
```

**这就是3DGS将3D高斯投影到2D的核心公式！**

### 投影均值

```
μ₂D = (fx · X₀/Z₀ + cx, fy · Y₀/Z₀ + cy)
```

```python

```

## 3.5 球谐函数 —— 3DGS的颜色表示

### 为什么需要球谐函数？

真实世界中，物体的颜色会随观察角度变化（视角相关外观）：
- 金属表面：正面看亮，侧面看暗
- 玻璃：不同角度反射不同颜色
- 建筑外墙：不同光照方向下颜色不同

如果直接存储RGB颜色，无法表示这种视角相关性。

**解决方案**：用球谐函数（Spherical Harmonics）表示颜色关于视线方向的函数。

### 球谐函数的定义

球谐函数 Yₗᵐ(θ, φ) 是定义在球面上的正交基函数，类似于傅里叶级数在圆上的作用。

**前几阶球谐函数**：
- 0阶（1个）：Y₀⁰ = 1/√(4π)（常数，各向同性）
- 1阶（3个）：Y₁⁻¹, Y₁⁰, Y₁¹（线性，表示方向性）
- 2阶（5个）：Y₂⁻²,...,Y₂²（二次，表示更复杂的方向性）
- 3阶（7个）：Y₃⁻³,...,Y₃³（三次）

**总计**：0-3阶共 (3+1)² = 16 个基函数

### 用球谐函数表示颜色

对于每个高斯基元，颜色是视线方向 d 的函数：

```
c(d) = Σₗ Σₘ cₗᵐ · Yₗᵐ(d)
```

其中 cₗᵐ 是球谐系数（每个系数对应RGB三个通道）。

**3DGS中的参数**：
- `_features_dc`：0阶系数（1×3，DC项，各向同性颜色）
- `_features_rest`：1-3阶系数（15×3，视角相关部分）

**与官方代码的对应**：
```python
# utils/sh_utils.py
def eval_sh(deg, sh, dirs):
    """
    计算球谐函数值
    deg: 最高阶数
    sh: 球谐系数 (N, (deg+1)², 3)
    dirs: 视线方向 (N, 3)
    """
    # 0阶
    result = C0 * sh[:, 0]
    if deg > 0:
        x, y, z = dirs[:, 0:1], dirs[:, 1:2], dirs[:, 2:3]
        result += -C1 * y * sh[:, 1] + C1 * z * sh[:, 2] - C1 * x * sh[:, 3]
    # ... 更高阶
    return result
```

### 球谐函数的正交性

球谐函数满足正交归一性：
```
∫ Yₗᵐ(θ,φ) · Yₗ'ᵐ'(θ,φ) dΩ = δₗₗ'δₘₘ'
```

这意味着不同阶的球谐函数互相独立，可以独立优化。

## 常见误区

**误区1**：高斯分布的参数越多越好
- 不一定！3DGS用3阶球谐（16个系数），更高阶会过拟合且参数太多

**误区2**：3D高斯投影到2D后不再是高斯
- 严格来说是对的（透视投影是非线性的）
- 但3DGS用雅可比近似，在实践中误差很小

**误区3**：协方差矩阵的行列式越大，高斯越"大"
- 不完全对！行列式是所有特征值的乘积，反映的是"体积"
- 一个细长的高斯（一个特征值大，其他小）行列式可能很小
