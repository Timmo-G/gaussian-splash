# Gaussian Splash Algorithm Learning
A step-by-step learning project for Gaussian Splash Algorithm with Python, divided into 3 phases (foundation -> core implementation -> optimization).  

## Project Overview
This repository records the learning process of Gaussian Splash Algorithm, including:
- Phase 1: Python/NumPy/Matplotlib foundation for Gaussian calculation & visualization
- Phase 2: Core function implementation of Gaussian Splash (parameterization, multi-splash overlay, modularization)
- Phase 3: Advanced optimization & expansion (GUI, performance acceleration, 3D visualization, etc.)

## Folder Structure
| Folder Name                | Description                                  |
|----------------------------|----------------------------------------------|
| 01_foundation_phase        | Basic Python/numerical calculation & 2D visualization |
| 02_core_implementation     | Core Gaussian Splash function & modularization |
| 03_optimization_expansion  | Advanced optimization and extended functions |
